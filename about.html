About this web page

Even tho i have wide range of "weaponry" at my disposal i decided to make this website as basic as possible.

  • Using only native JavaScript, without any libraries, plugins or frameworks.
  • Does not contain a single image or inline svg.
  • Every element is made only by using CSS, cross-browser supported and widely used.
  • This website is made only as a showcase, main focus was on quality of code and browser performance.
  • Made mainly for desktop view due to wider range of events.
  • A "scalable" website without ordinary pixel-limited containers.
